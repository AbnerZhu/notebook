# TOEFL

## Words

**如何记忆单词**

- 阅读记忆
  - [economist](http://www.economist.com)
  - [times](www.times.com)
  - [gardian](www.gardian.co.uk)
  - 生词圈出， 反复研读句子。 勾画词组， 了解替换， 熟词僻意
- 词根、 词缀、 词源、 字母学规律
  - 词根词缀 
    - prefix
      - 肯定， 否定
      - 方向
      - 数字
    - suffix: 词性
    - root: 意义（根的逆推）
  - 词源 Etymology （看读音， 看拼写， 看意思）
  - 词源的变化规律
    - 元音之间可以互换： a, e, i, o, u, y
    - 辅音之间 p/b, t/d, k/g/c/qu/x, f/v, s/z/x/th 可以互换
    - 形近字母的互换: u/v/w, m/n 的互换
    - 字母g, h 的脱落 (不发音)
- 联想
  - 单音节词汇 —— 形近词
    - fiend <—> friend
    - spout <—> sprout
  - 多音节词汇要拆词 —— 要拆成认识的词根词缀， 拼写， 熟词和与熟词形近的词等
- 复习（短时多次）
  - 0 —> 0.5H —> 2.0H —> <4.0H
  - D1  L1(3)
  - D2  L1 L2
  - D3  L3 L2
  - D4  L1 L3 L4
  - D5  L5 L2 L4
  - D6  L6 L5 L3
  - D7  L1 L4 L6 L7
  - D8  L2 L3 L5 L8
